Learn AI Recommendation Systems Through Minecraft

BeeTrap is a Minecraft-based educational game that uses the analogy of bee pollination to teach kids about filter bubbles in AI recommendation systems. As players pollinate flowers, similar ones grow; visually demonstrating how algorithms can trap users in a cycle of sameness.

Players learn to break filter bubbles by applying a diversification strategy, all while immersed in an interactive, environmental science-inspired exploratory environment.

BeeCurious builds on BeeTrap by adding an embodied conversational agent called Bip; a bee-like character that talks and interacts with players during gameplay. It encourages reflection, asks questions, and helps make complex AI ideas more relatable.

Together, BeeTrap and BeeCurious aim to boost AI literacy, ethics, and critical thinking through embodied, playful learning.

üì¶ Usage Instruction

Follow these step-by-step instructions to install BeeCurious and start your AI learning journey:

  1. Install Git (if needed)
  2. Java 17 or higher (Required for Minecraft 1.21.4)
  3. Clone the project
  4. Download the world
    • Visit the GitHub Releases page
    • Download the .zip file
    • Export the zip file's content under run/saves folder within the cloned project (make sure there are no redundant parent folders)
  5. Configure your OpenAI API and Typecast API key (if using Bip) by editing run/.env file
  6. Launch and Play choose beetrap world
  7. Enter the game by entering ```/game new 3```
  8. Start exploring and learning about AI filter bubbles
  9. Clean up your world for next time by entering ```/game destroy```
