Learn AI Recommendation Systems Through Minecraft

BeeTrap is a Minecraft-based educational game that uses the analogy of bee pollination to teach kids about filter bubbles in AI recommendation systems. As players pollinate flowers, similar ones grow; visually demonstrating how algorithms can trap users in a cycle of sameness.

Players learn to break filter bubbles by applying a diversification strategy, all while immersed in an interactive, environmental science-inspired exploratory environment.

BeeCurious builds on BeeTrap by adding an embodied conversational agent; a bee-like character that talks and interact with players during gameplay. It encourages reflection, asks questions, and helps make complex AI ideas more relatable.

Together, BeeTrap and BeeCurious aim to boost AI literacy, ethics, and critical thinking through embodied, playful learning.

üì¶ Installation Guide

Follow these step-by-step instructions to install BeeCurious and start your AI learning journey:

Prerequisites

  1. Minecraft Java Edition 1.21.4 - Make sure you have the correct version installed
  2. Fabric Loader - Download and install from fabricmc.net
  3. Fabric API - Download the latest version compatible with 1.21.4
  4. Java 17 or higher - Required for Minecraft 1.21.4

Installation Steps

  1. Download the Mod
    • Visit the GitHub Releases page
    • Download the latest .jar file
  2. Install Fabric Loader
    • Run the Fabric installer
    • Select Minecraft version 1.21.4
    • Click "Install"
  3. Add Mods to Minecraft
    • Navigate to your Minecraft directory
    • Windows: %appdata%\.minecraft\mods
    • macOS: ~/Library/Application Support/minecraft/mods
    • Linux: ~/.minecraft/mods
    • Place both the BeeCurious mod file and Fabric API in the mods folder
  4. Configure AI Features (Optional)
    • For full conversational AI features, you'll need an OpenAI API key
    • The mod will work without it, but with limited AI interactions
    • Configuration details are available in the development guide
  5. Launch and Play
    • Start Minecraft Launcher
    • Select the Fabric profile
    • Create a new world or join an existing one
    • Start exploring and learning about AI filter bubbles!

Troubleshooting

  • Mod not loading: Ensure you have the correct Minecraft version (1.21.4) and Fabric API
  • Crashes on startup: Check that all dependencies are installed and compatible
  • AI features not working: Verify your OpenAI API configuration
  • Performance issues: Allocate more RAM to Minecraft (recommended: 4GB+)
